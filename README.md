
# Ark Lite Wallet

## Development

```
git clone https://github.com/ArkEcosystem/ark-lite-wallet
cd ark-lite-wallet
npm install
cd src
npm run dev
```

Open http://localhost:8080

## Build

```
npm run build
```

## Electron

### Start

Start the Electron client.

```
npm run start
```

### Windows

Build package for Windows.

```
npm run dist:win
```

### Mac OS X

Build package for Mac OS X.

```
npm run dist:osx
```

### Linux

Build package for Linux.

```
npm run dist:linux
```
